## 🚀 3-STEP QUICK START

### Step 1: Install MongoDB
```bash
# Download from: https://www.mongodb.com/try/download/community
# Run MSI installer → Complete
# Verify: mongosh
```
→ See `MONGODB_LOCAL_SETUP.md` for details

### Step 2: Install Dependencies
```bash
# Frontend
cd F:\Thesis\frontend
npm install

# Backend
cd F:\Thesis\backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### Step 3: Run Your App
```bash
# Terminal 1 - Backend
cd F:\Thesis\backend
venv\Scripts\activate
python -m app.main

# Terminal 2 - Frontend
cd F:\Thesis\frontend
npm run dev
```

**Access:**
- Frontend: http://localhost:3000
- Backend: http://localhost:8000
- API Docs: http://localhost:8000/docs
